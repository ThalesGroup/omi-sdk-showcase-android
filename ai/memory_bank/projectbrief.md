# Project Brief

## Project Name
Showcase Thales (OneWelcome SDK Showcase)

## Description
This is an Android application designed to demonstrate and showcase the features and capabilities of the OneWelcome (Thales) Mobile SDK. It serves as a reference implementation for developers integrating the SDK into their own applications.

## Core Goals
1.  **Demonstrate Features:** Provide clear, working examples of SDK functionality (Authentication, Registration, Mobile Auth, etc.).
2.  **Reference Implementation:** Show best practices for integrating the SDK using modern Android architecture.
3.  **Testing Tool:** Allow internal teams and clients to test SDK features in a controlled environment.

## Key Features
-   User Registration (Browser, Two-step)
-   User Authentication (PIN, Biometric, Implicit)
-   Mobile Authentication (Push, OTP)
-   Change PIN
-   Deregistration
-   Resource Access
