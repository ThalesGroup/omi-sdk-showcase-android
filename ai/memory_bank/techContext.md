# Tech Context

## Technology Stack

### Language
-   **Kotlin:** Primary programming language.

### UI Framework
-   **Jetpack Compose:** Modern toolkit for building native UI.
-   **Material Design 3:** Design system implementation.

### Architecture Components
-   **ViewModel:** State management.
-   **Navigation Compose:** Handling in-app navigation.
-   **Hilt:** Dependency Injection.
-   **Lifecycle:** Managing Android lifecycle events.

### Concurrency
-   **Coroutines:** Asynchronous programming.
-   **Flow:** Reactive data streams (StateFlow, SharedFlow).

### Build System
-   **Gradle:** Build automation (Kotlin DSL `build.gradle.kts`).

### External Dependencies
-   **OneWelcome (Thales) SDK:** The core library being showcased.
-   **Firebase (FCM):** Likely used for Push Mobile Authentication features.

## Development Environment
-   **IDE:** Android Studio.
-   **JDK:** JDK 17 or higher recommended.
-   **Android SDK:** Target SDK likely 34+.
