# Product Context

## Problem Statement
Developers integrating the OneWelcome SDK need a clear, functional reference to understand how to implement various authentication and identity management features. Without a showcase app, integration can be error-prone and time-consuming.

## Solution
The Showcase App provides a tangible "playground" where developers can see the SDK in action. It covers the entire lifecycle of a user's identity on a mobile device, from registration to authentication and deregistration.

## User Experience
The app is designed with a developer-centric UX:
-   **Feature-based Navigation:** Features are grouped logically (Registration, Authentication, etc.).
-   **Clear Feedback:** Operations provide visual feedback (Success/Failure) and logs.
-   **Configuration:** Allows runtime configuration of the SDK connection details.

## Domain
-   **Mobile Security:** Secure authentication, biometrics, PIN management.
-   **Identity Management:** User profiles, device registration.
-   **OIDC/OAuth:** Standard protocols for authentication flows.
